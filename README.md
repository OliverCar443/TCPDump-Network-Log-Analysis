# TCPDump Network Log Analysis

## Objective
The goal is to **analyze traffic in transit** using data from a network protocol analyzer tool (TCPDump) and **identify** which network protocol and service were impacted by this incident.
<br>


**(Note)** In the internet layer of the TCP/IP model (Network Layer 2), the IP formats data packets into IP datagrams. The information provided in the datagram of an IP packet can provide security analysts with insight into suspicious data packets in transit. Knowing how to identify potentially malicious traffic on a network can help cybersecurity analysts assess security risks on a network and reinforce network security.
<br>
